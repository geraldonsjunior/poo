MÓDULO 1: INTRODUÇÃO À PROGRAMAÇÃO

Configuração do ambiente e primeiro programa

1.1 O que é Programação?

Programação é como dar instruções passo a passo para o computador, similar a uma receita de bolo:

1. Pegue duas fatias de pão
2. Passe manteiga em uma fatia
3. Coloque o queijo
4. Leve ao forno por 5 minutos

1.2 Configurando o VS Code

Passos para instalação:

1. Baixe e instale o JDK 17+

2. Instale o VS Code

3. Instale a extensão "Extension Pack for Java"

4. Verifique a instalação no terminal: java -version

1.3 Primeiro Programa: Hello World

Crie um novo projeto no VS Code e digite:

// App.java
public class App {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        System.out.println("Meu primeiro programa!");
    }
}

Explicação:

• public class App: Define uma classe

• main: Método principal onde o programa começa

• System.out.println(): Exibe texto na tela

MÓDULO 2: FUNDAMENTOS DO JAVA

Variáveis, operadores e entrada de dados

2.1 Variáveis e Tipos de Dados

Variáveis são como "caixinhas" que armazenam informações:

// Tipos primitivos
int idade = 16;
double altura = 1.75;
char letra = 'A';
boolean ativo = true;

// String (não é primitivo)
String nome = "Maria";
String escola = "Escola Técnica";

2.2 Operadores

Operadores Aritméticos: + - * / %

Operadores Relacionais: == != > < >= <=

Operadores Lógicos: && || !

// Exemplo prático
int a = 15;
int b = 4;
System.out.println("Soma: " + (a + b));
System.out.println("Resto: " + (a % b));

2.3 Entrada de Dados com Scanner

import java.util.Scanner;

public class EntradaDados {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("Digite seu nome: ");
        String nome = scanner.nextLine();
        
        System.out.print("Digite sua idade: ");
        int idade = scanner.nextInt();
        
        System.out.println("Olá " + nome + ", você tem " + idade + " anos!");
        
        scanner.close();
    }
}

MÓDULO 3: ESTRUTURAS DE CONTROLE

Condicionais e loops

3.1 Estruturas Condicionais (if/else)

// Verificação de idade
if (idade >= 18) {
    System.out.println("Maior de idade!");
} else {
    System.out.println("Menor de idade!");
}

// Sistema de notas
if (media >= 7.0) {
    System.out.println("Aprovado!");
} else if (media >= 5.0) {
    System.out.println("Recuperação!");
} else {
    System.out.println("Reprovado!");
}

3.2 Estruturas de Repetição (Loops)

Loop FOR - Quando sabemos quantas vezes repetir:

// Contagem de 1 a 10
for (int i = 1; i <= 10; i++) {
    System.out.println("Número: " + i);
}

// Tabuada do 5
for (int i = 1; i <= 10; i++) {
    System.out.println("5 x " + i + " = " + (5 * i));
}

Loop WHILE - Quando NÃO sabemos quantas vezes repetir:

// Adivinhação de número
int numeroSecreto = 42;
int palpite = 0;

while (palpite != numeroSecreto) {
    System.out.print("Seu palpite: ");
    palpite = scanner.nextInt();
    
    if (palpite < numeroSecreto) {
        System.out.println("Muito baixo!");
    } else if (palpite > numeroSecreto) {
        System.out.println("Muito alto!");
    }
}
